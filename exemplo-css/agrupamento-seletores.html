Agrupamento de Seletores

Podems aplicar as mesmas regras CSS para vários seletores diferentes. Para isso, podemos separar os seletores utilizando uma vírgula.

Teste

Assim, não é necessário copiar e colar os mesmos códigos várias vezes. É possível fazer um único bloco de CSS e reutiizar as mesmas regras.
